MATLAB - Variables

❮ Previous Next ❯

Variables are letters or words that have been assigned a value, such as numbers or arrays (more about those later!).

This value can then be recalled for later use in calculations. Current variable values can be seen down the right hand side of MATLAB under "Workspace".

Example

Values can simply be assigned by declaring : [Variable Name] = [Value]

a = 2; b = 7; c = a * b
c = 14

Who

The who command displays all of the variable names used.

Your variables are:
a b c

Clear

The clear command displays all current variable names in use.
