MATLAB - Loops

There may be occasions where you need to execute a section of code multiple times. This is done though creating a loop.

There are three types of loops: While Loop, For Loop and Nested loops.

While Loop

These loops continue to execute the code inside of it while a set condition is met. 
a = 10;                
while( a < 20 )
    fprintf('value of a: %d\n', a);
    a = a + 1;
end
Here the code will print out the value of a, while a is smaller than 20, producing the following result - 
value of a: 10
value of a: 11
value of a: 12
value of a: 13
value of a: 14
value of a: 15
value of a: 16
value of a: 17
value of a: 18
value of a: 19

For Loop

These loops execute code for a pre determined amount of time or steps. There are three conditions that can be set -
  • [initial value:end value]
    • for a = 10:20
    fprintf('value of a: %d\n', a);
end

    This increments the variable by 1 from initial value to end value, ending when the variable a is greater than end value. Therefore this displays every value between 10 and 20, inclusive.

    value of a: 10
value of a: 11
value of a: 12
value of a: 13
...
value of a: 17
value of a: 18
value of a: 19
value of a: 20
  • [initial value]:[step]:[end value]
    • for a = 10: 2: 20
    disp(a)
end

    This is the same as rhe previous loop, but you can specify how much to increment by each time. In this example, the value grows by 2 each time. This value can also be negative. The outcome of this loop would be -

    value of a: 10
value of a: 12
value of a: 14
value of a: 16
value of a: 18
value of a: 20
  • [value array]
    • for a = [24,18,17,23,28]
    a*2
end

    This loop cycles through all items within an array, executing the code with the value set to that variable. This example would produce all items within the array, but times by 2. To display the values in the array, use the disp operator

    ans = 48
ans = 36
ans = 34
ans = 46
ans = 56

Nested Loop

Nested loops are simply loops within other loops, with each expression being the conditions that must be met, such as a while or for loop, and the statement is the code that will be executed. 
while [expression1]
    for [expression2]
        [statement]
    end
end
