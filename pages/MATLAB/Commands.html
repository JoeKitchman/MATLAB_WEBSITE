❮ Previous Next ❯

MATLAB has commands to help make working within it much easier. We covered some of these in the previous page such as who and clear but there are a few more essential ones you should know.

Session Commands

These commands are used for managing a current session -

Command Purpose
clear Removes variables from memory
who Lists current variable names.
clc Clears command window.
quit Stops MATLAB.
lookfor Searches help entries for a keyword.
