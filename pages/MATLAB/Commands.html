MATLAB - Commands

❮ Previous Next ❯

MATLAB has commands to help make working within it much easier. Commands are executed within the command window.

These commands are used for managing a current session. They are some of the most basic but most useful commands and will see them used often throughout this tutorial in the videos.

Command Purpose
clear Removes variables from memory
who Lists current variable names.
clc Clears command window.
quit Stops MATLAB.
lookfor Searches help entries for a keyword.
