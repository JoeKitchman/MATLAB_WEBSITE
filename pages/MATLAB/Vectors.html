MATLAB - Vectors

❮ Previous Next ❯

A vector is a one-dimensional array of numbers. MATLAB allows creating of vectors in two ways, row or column vectors. Unlike some programming languages, arrays and vectors in MATLAB do not start at 0.

Row Vectors

Row vectors are created by surrounding the set of numbers in square brackets, using spaces or commas to define each of the elements. 

x = [1, 2, 3, 4, 5]

Column Vectors

Row vectors are created by surrounding the set of numbers in square brackets, using semicolons> to define each of the elements. 

x = [1; 2; 3; 4; 5]

Referancing elements

You can obtain specific elements within a vector or array in multiple ways, by specifing a location within the vector, a range or the entire vector.

The ith component of a vector v is referred as v(i).

x = [5, 6, 7, 8, 9]
x(3)
ans = 7

In this example the 3rd element of the vector has been specified and in this case, is 7

A range within the vector can also be used to identify elements.

x = [5, 6, 7, 8, 9]
sub_x = x(2:4)
sub_x = 
    6 7 8

Vector Manipulation

Addition and Subtraction

You can add or subtract two vectors together. Both the vectors must be of same type (row and row, column and column) and contain same number of elements.

x = [1, 2, 3, 4, 5]
y = [4, 7, 12, 6, 9]
z = x + y 
disp(z)

Adding these two vectors together will produce the following result:

5 9 15 10 14

Appending Vectors

MATLAB has the power to allow two vectors, of the same length and type, to be put together to create a new one.

Two row vectors can be appended with:

r = [r1,r2]

Two column vectors can be appended with:

c = [c1;c2]

This example shows two row vectors appended together to create a new vector r:

r1 = [1, 2, 3, 4];
r2 = [5, 6, 7, 8];
r = [r1,r2];
disp(r)
r = 1 2 3 4 5 6 7 8

Dot Product

The dot product of two vectors, again of equal length and format, can be calcultaed using the dot() function. 

v1 = [1, 2, 3, 4];
v2 = [5, 6, 7, 8];
dp = dot(v1, v2);
disp(dp);
dp = 70
